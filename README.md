# Smart Construction Frontend

基于 Three.js 的智慧工地 3D 可视化系统。

## 功能特性

- Three.js 3D渲染
- 工地场景可视化
- 交互控制（旋转、缩放、平移）

## 技术栈

- Three.js
- JavaScript (ES6+)
- HTML5/CSS3

## 目录结构

```
smart-construction-frontend/
├── index.html      # 主页面
├── main.js         # 主逻辑
└── libs/           # 第三方库
    ├── three.module.js
    └── OrbitControls.js
```

## 使用方法

直接在浏览器中打开 `index.html`，或使用本地服务器：

```bash
python -m http.server 8080
```

然后访问 http://localhost:8080
